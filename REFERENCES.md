# References

- [Typst](https://github.com/typst/typst) - A markup-based typesetting system.
- [CeTZ](https://github.com/cetz-package/cetz) - A library for drawing with Typst.
- [typst-fletcher](https://github.com/Jollywatt/typst-fletcher) - Typst package for drawing diagrams with arrows, built on top of CeTZ.
- [Primer Design System](https://primer.style/) - The design system for GitHub.
- [GitHub Brand Toolkit](https://brand.github.com/)
- [Nerd Fonts](https://www.nerdfonts.com/) - patches developer targeted fonts with a high number of glyphs (icons).
- [Nerd Fonts Wiki](https://github.com/ryanoasis/nerd-fonts/wiki)
- [SVG Shapes](https://www.shapes.gallery/)
